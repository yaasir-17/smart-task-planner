# Study Planner

A web application designed to help students organize study schedules, track tasks, and manage academic goals.

## Planned Features

* User authentication
* Study task management
* Daily and weekly study schedules
* Progress tracking
* Deadline reminders
* Dashboard with study statistics

## Technologies

* Python
* Flask
* SQLite
* Bootstrap
* HTML/CSS
* Jinja2

## Status

Currently under development.
